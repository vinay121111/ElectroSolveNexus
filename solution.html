    • Step-by-Step Guifor Battery Optimization on Windows
    1. Use the Built-in Troubleshooter

    1. Click on the taskbar search iconand type "troubleshooter settings" .
    2. Click on "Other troubleshooters"
    3. Find "Power" and click on "Run" next to it.
    2. Change the Power Plan
    1. 1. Press Windows key + I to open Settings.
    2. 2. Go to "Power & battery".
    3. 3. Scroll down to "Power mode".
    4. 4. Select "Best power efficiency" from the dropdown menu.
    3. Enable Battery Saver
    1. Press Windows key + I to open Settings.
    2. Go to "Power & battery".
    3. Expand the "Battery saver" section.
    4. Click on "Turn on now".
    5. Optionally, set it to turn on automatically at a certain battery level.
    4. Optimize Display Settings
    1. Press Windows key + Ato open quick settings.
    2. Use the slider next to brightness to reduce the screen brightness.
    5. Modify Video Playback Settings
    1. Open Settings and go to the "Apps" tab.
    2. Select "Video playback".
    3. Choose "Optimize battery life" under battery options.
    4. Optionally, select to play videos at a lower resolution when on battery.
    6. Ensure Background Apps Aren't Draining Battery
    1. Press Windows key + I to launch Settings.
    2. Go to "Power & battery".
    3. Under "Battery", select "Battery usage".
    4. Review the apps using the most battery and turn off background activity for unimportant apps.
    7. Disable Fast Startup
    1. Press Windows key + R to open the Run dialog.
    2. Type "control"and press Enter.
    3. Change the view type to icons and select "Power options".
    4. Click on "Choose what power buttons do" on the left sidebar.
    5. Click on "Change settings that are currently unavailable".
    6. Uncheck the box that reads "Turn on fast startup" .
    8. Check Battery Health
    1. Press Windows key + X and select "Windows Terminal (Admin)".
    2. Type in the command `powercfg /batteryreport` and press Enter.
    3. This will generate a battery report that you can analyze to see if the battery needs replacing.

    Steps to Fix Missing Default Apps on Windows 11
    1. Use Windows Troubleshooter

      Description:Use the built-in Windows troubleshooter to diagnose and fix missing default apps.
    1. Press Windows key + Ito open Settings.
    2. Go to System > Troubleshoot > Other troubleshooters.
    3. Find and run the Windows Store Apps troubleshooter.
    2. Repair Individual Apps
      Description: Repair individual apps that are missing.
    1. Right-click on the Start icon and select Settings.
    2. Navigate to the Apps tab and click on Apps & features.
    3. Locate the app you want to repair, click on the three-dot menu icon next to it.
    4. Go to Advanced options and click on Repair.
    3. Reinstall Apps Using Windows PowerShell
      Description: Reinstall missing apps using a PowerShell command.
    1. Right-click on the **Start** icon and select **Windows Terminal (Admin)**.
    2. Copy and paste the following command into the PowerShell console:
      powershell Get-AppxPackage -allusers Microsoft.WindowsStore | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}
    3. Press Enter to execute the command.
    4. Boot Your System in a Clean Boot State
      Description: Diagnose if a third-party app is interfering with Windows by booting in a clean boot state.
    1. Press Windows key + R, type `msconfig`, and press Enter. >
    2. Go to the Services tab, check Hide all Microsoft services, and click on Disable all.
    3. Go to the Startup tab and click on Open Task Manager.
    4. Disable all startup items in Task Manager.
    5. Close Task Manager and click OK in the System Configuration window.
    6. Restart your PC.
    5. Create a New User Account
      Description:Create a new user account to check if the issue is related to your current user account.
    1. Press Windows key + I to open Settings.
    2. Go to Accounts > Family & other users.
    3. Click on Add account under Other users.
    4. Follow the prompts to create a new user account.
    5. Sign in with the new user account and check if the missing apps appear.
    6. Reset Your PC
      Description: Reset your PC to remove installed apps and reset system settings while keeping personal data.
    1. Press Windows key + I to open Settings.
    2. Go to System > Recovery.
    3. Under Reset this PC, click on Reset PC.
    4. Choose Keep my filesand follow the prompts to complete the reset.

    Steps to Fix Crashing Apps on a Samsung Phone

      Solution 1: Uninstall Updates to the Android System Webview App Open Settings:
    1. Open the Settings app on your Samsung phone. Go to Apps: Scroll down and tap on Apps.
    2. Find Android System Webview: Scroll through the list or use the search bar at the top to find Android System Webview.
    3. Uninstall Updates: Tap on Android System Webview to open its settings. Tap on the three-dot menu icon at the top right corner. Select Uninstall updates. Confirm the action if prompted.
      Solution 2: Disable the Android System Webview App Open Settings:
    1. Open the Settings app on your Samsung phone. Go to Apps: Scroll down and tap on Apps.
    2. Find Android System Webview: Scroll through the list or use the search bar at the top to find Android System Webview.

    Steps to Assemble a Computer

    1. Install the power supply into the case.
    2. Install the motherboard into the case.
    3. Install the CPU into the motherboard.
    4. Install the RAM into the motherboard.
    5. Install the hard disk into the case.
    6. Install the CD-ROM drive into the case.
    7. Connect all of the cables together.
    8. Close up the case and turn on the computer.

    Breakdown of some of the key mobile parts and their functions:

    1. Mainboard (also known as Motherboard or PCB - Printed Circuit Board): This is the heart of the phone. It connects all the other parts together and carries electrical signals.
    2. Central Processing Unit (CPU): This is the brain of the phone, responsible for processing information and running apps.
    3. Random Access Memory (RAM): This temporary storage holds data the phone is actively using for faster access.
    4. Storage (ROM): This is where all your data is stored permanently, like photos, videos, apps, and music.
    5. Battery: This provides power to the phone.
    6. Display: This is the screen you use to interact with the phone.
    7. Camera: This allows you to take pictures and videos.
    8. Microphone: This captures sound for calls and recordings.
    9. Speaker: This plays sounds and music.
    10. Sensors: These detect things like motion, orientation, and light.
    11. Cellular modem: This allows your phone to connect to cellular networks for calls and data.
    12. Wi-Fi module: This allows your phone to connect to Wi-Fi networks.
    13. Bluetooth module: This allows your phone to connect to other Bluetooth devices wirelessly.
    14. Charging port: This is where you connect the charger to power the phone.
    15. Headphone jack: This allows you to connect headphones to the phone (some phones may not have this).
